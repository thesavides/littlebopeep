## 6. Translation system

| Property | Value |
|---|---|
| **Cache version** | `v16` (in `src/lib/i18n.ts`) |
| **Cache key pattern** | `translations_v16_{lang}` in localStorage |
| **Cache TTL** | 24 hours |
| **Languages** | `en`, `cy` (Welsh), `ga` (Irish), `gd` (Scottish Gaelic) |
| **Seeding tool** | Node.js inline scripts using `@supabase/supabase-js` + Anthropic Haiku for auto-translation |

**To add new translations:**
1. Use `t('key.name', { optionalParams }, 'English fallback')` in the component
2. Run a Node seed script to insert rows into `translations` table for all 4 languages
3. Bump `CACHE_VERSION` in `src/lib/i18n.ts`
4. Commit + deploy

## 9. Known gaps / remaining work

1. **Farmer dashboard mobile menu (PPAP Req 11)**: BottomNav exists with Dashboard / Farms / Alerts / Profile tabs, but the UI isn't a true bottom-tab-bar pattern matching WalkerDashboard — layout still uses stacked sections rather than tab-switching views.

2. **Offline photo sync**: Photos captured offline (base64 in IndexedDB) are NOT uploaded to Supabase Storage on sync. Synced reports have `photoUrls = []`. Fix would require re-reading data URLs from IndexedDB and uploading to Storage during sync.

3. **Thank-you sent state persistence**: The "✓ Sent" indicator is local React state — resets on page reload. Would need a DB query on mount to check `notifications` table for existing `thank_you` rows for each report.

4. **Walker edit — photo replacement**: `editOwnReport` works for text fields; replacing/removing existing photos is not implemented.

5. **PWA icons**: `public/manifest.json` references `/icons/icon-192.png` and `/icons/icon-512.png` — these files do not exist. App installs but shows blank icon.

6. **`getLocalizedCategoryName()`**: Helper exported from `appStore.ts` but not yet wired into walker category picker or farmer subscription toggles.

## 10. Critical implementation rules

| Rule | Why |
|---|---|
| **Commit before deploy** | `.open-next/` is gitignored; deploying uncommitted work is untracked |
| **No `runtime = 'edge'`** | Breaks `@supabase/supabase-js` under Cloudflare edge runtime |
| **`supabase.auth.admin` is server-only** | Needs service role key — use `/api/admin/` routes |
| **Bump `CACHE_VERSION` after seeding** | Stale localStorage cache will serve old translations for 24h |
| **Refresh session before offline sync** | JWT expires offline; `autoRefreshToken` is async — race condition causes 401 |
| **`condition` field is free text** | No CHECK constraint (dropped migration 006). `conditions[]` is the multi-select array |
| **Farmer dropdowns include admins** | Pass `[...farmers, ...admins]` as owner list in farm/claim modals |
| **z-index layering** | HelpChatbot FAB: z-50. All full-screen overlays (OfflineCapture, modals): z-[60] or higher |
